# Arduino_Tone
通过arduino操控蜂鸣器演奏音乐


就是通过ardunio 操控蜂鸣器，发出音乐。
==================================
以 arduino uno 为例，

需要：蜂鸣器一个

   公对母杜邦线 2 条


接线方式 ：蜂鸣器正极接 digit 8 号。
          
    蜂鸣器负极接 digit GND 
    
  ==============================  
    

演示方式：将 pitchesNMB.h 和 xxx.ino 一起放入 名为 ino 文件的同名文件夹，

        打开电脑，用 Arduino IDE 打开需要播放的 ino 文件，【验证】无误后，执行【上传】就可以听到乐曲播放了。
